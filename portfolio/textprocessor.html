Description

The Textpreprocessor Project is the tool for processing various text strings.

Features

  • cleaning code from html tags to extract simple text
  • cleaning multiple spaces, tabs and linebreaks from text
  • deleting repeating words
  • reducing text from words that are including in otehr words this text
  • capitalizing first letter of certains word
  • inserting special characters on special position in text
  • ... some other features in plans
